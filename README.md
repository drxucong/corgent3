# Corgent Holdings — Official Website

Bilingual (English / 中文) website for **Corgent Holdings Limited**, a Hong Kong–based
test-fixture and electronic-design consultancy.

Pure static HTML/CSS/JS — no build step required.

## Local preview

```bash
python -m http.server 8099
# open http://localhost:8099
```

## Structure

```
index.html        — main single-page site
privacy.html      — privacy policy
terms.html        — terms of use
assets/css        — stylesheet (design system)
assets/js         — language toggle, navigation, animations, form
assets/img        — logo and photography
```
